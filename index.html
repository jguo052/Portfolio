Natural Language Processing

I pulled approximately 5,000 forum posts from Reddit's API (specifically from the subreddits r/Seattle and r/SeattleWA). After cleaning the text within each post, I identified major language differences between the two subreddits through ranking each word. Finally, I constructed an ensemble model to predict which subreddit a post would likely originate from given the language used.

Housing Prices
